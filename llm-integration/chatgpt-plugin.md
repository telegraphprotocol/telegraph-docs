# ChatGPT Plugin

Telegraph's LLM plugin allows ChatGPT users to easily access blockchain data from numerous chains via natural language. From creating a new wallet, to performing a crosschain transaction, Telegraph's plugin can be used for a wide variety of interactions and data investigation.

Telegraph plugin demo

The plugin is also structured to operate directly on the Telegraph site without the need for direct ChatGPT access. This will allow anyone to take advantage of the many benefits LLMs offer without needing to sign up for external accounts.

In order to sustain open access to all users, transactions that run through the plugin will have a small percentage of the native coin taken as a fee.
