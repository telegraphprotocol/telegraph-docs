# Validators

A validator is the moniker given to participants within the network that have the authorization to sign for requested crosschain transfers. Nodes will only respect incoming requests from other approved validators within the node network through the use of a multilayered system of checks that range from outright refusing messages at the load balancer level to requiring a signature to verify that the request truly stemmed from an authorized validator.
